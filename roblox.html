SOULBLOX (Combat Demo)

Genre: Multiplayer Action Adventure
Development Time: ~1 month
Team Size: Solo Developer

Role:
Programmer • Modeler • Animator • VFX Artist

Designed and implemented the entire prototype from scratch, including gameplay systems, animations, VFX, 3D assets, and scripting.

Core Features

  • 4-hit light attack combo
  • Directional block (180° frontal guard)
  • Parry system
  • 4-direction dash
  • Flash Step (temporary invisibility and speed boost)
  • Projectile-based ability system
  • Stat-scaled gameplay values (damage, speed, durations)
  • Custom Input Buffering Service

Systems Developed

  • Combat System
  • Ability Framework
  • Projectile System
  • Character Stats Integration
  • Animation Logic
  • VFX and Visual Feedback

Additional Features Not Fully Shown

  • Attack clash system
  • Enhanced dash effects
  • Animation polish
  • Punch-based attacks
  • Additional balancing and tuning
  • And more

All gameplay actions scale dynamically with character stats. For example, attack damage and Flash Step speed/duration are driven by the player's attributes.
